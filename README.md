| Layer | Technologies |

| Frontend | HTML5, CSS3, JavaScript |

| Databases | MySQL, MongoDB |
| Tooling | DOM API, localStorage, external REST APIs |

---

## Portfolio Features

- **Bilingual (i18n)** — Full English and Portuguese (pt-br) translation system, with language preference saved to `localStorage`
- **Dark / Light Theme** — One-click theme toggle with persistent state across sessions
- **Project Modals** — Accessible modal system with focus trapping, keyboard navigation (`Escape` to close), and screen-reader support
- **Stack Filtering** — Filter pills to browse projects by technology, powered by `data-*` attributes
- **Responsive Design** — Fluid grid layout with mobile breakpoints for all screen sizes
- **Sticky Navigation** — Frosted-glass navbar with `backdrop-filter` that stays in view while scrolling
- **Animated Footer Divider** — Accent line that expands to full width on hover via a cubic-bezier transition
- **Floating CTA Button** — Fixed contact button with a pulsing green availability indicator
